#  RateList Web App

RateList Web App is a simple and responsive web application designed to manage and display item price lists in a clean and user-friendly interface.

##  Features
- Responsive design for all devices  
- Clean and modern UI  
- Easy-to-view price list layout  
- Smooth user experience with interactive elements  

##  Technologies Used
- HTML  
- CSS  
- JavaScript  

##  Project Overview
This project is built to practice front-end development skills and improve UI/UX design understanding. It focuses on creating a structured and readable rate list system where users can easily view item prices in an organized way.

##  Purpose
The main goal of this project is to strengthen front-end development skills and build a practical web application that demonstrates real-world UI design concepts.
